第一章 正则表达式字符匹配攻略

2019-10-01 读书笔记 《JavaScript正则迷你书》 0 评论
字符组组合含义
/d[0-9]表示是一位数字
/D[^0-9]除数字外的任意字符
\w[0-9a-zA-Z_]表示数字、大小写字母和下划线
\W[^0-9a-zA-Z_]非单词字符
\s[ \t\v\n\r\f]空白符，包括空格、水平制表符、垂直制表符、换行符、回车符、换页符
\S[^ \t\v\n\r\f]非空字符
.[^\n\r\u2028\u2029]通配符，表示几乎任意字符。换行符、回车符、行分隔符和段分隔符除外
量词 {m,n} 通过在量词后面加个问号就能实现惰性匹配
  • {m,} 表示至少出现m次。
  • {m} 等价于{m,m}，表示出现m次
  • ? 等价于{0,1}
  • + 等价于{1,}
  • * 等价于{0,}
